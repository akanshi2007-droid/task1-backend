# Campus Lost & Found API
## 📌 Description
This project is a REST API built using Node.js and Express that allows users to manage lost and found items on campus.

## Features
1.Add lost/found items
2.View all items
3.Search items by title
4.Filter items by type (lost/found)
5.Update item details
6.Delete items
7.Input validation and error handling
  
## Tech Stack
* Node.js
* Express.js
* JSON file storage

## API Endpoints
1.GET /items
2.POST /items
3.PUT /items/:id
4.DELETE /items/:id

## How to Run
npm install
node index.js

Server runs on:
http://localhost:5000
